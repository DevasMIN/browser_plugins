# SpeedPlayBack - Video Speed Controller ⚡

Firefox расширение для управления скоростью воспроизведения видео с настраиваемыми пресетами и горячими клавишами.

> **Поддержка языков:** 🇷🇺 Русский | 🇺🇸 English | 🇪🇸 Español

## Установка

### Временная установка (для тестирования)
1. Откройте Firefox и перейдите в `about:debugging#/runtime/this-firefox`
2. Нажмите "Load Temporary Add-on..."
3. Выберите файл `manifest.json` из папки `video_speed_controller_firefox`

### Постоянная установка
1. Упакуйте расширение в .xpi файл
2. Подпишите расширение в Mozilla Add-ons
3. Установите через Firefox Add-ons Manager

## Icon


You can use any graphic editor or online icon generator.

## Features

- ✅ Speed control through buttons and presets
- ✅ Edit preset cells (click the ✎ icon)
- ✅ Assign keyboard shortcuts for each preset
- ✅ Three application scopes: for tab, domain, or all tabs
- ✅ Enable/disable extension
- ✅ Reset all settings
- ✅ Automatic settings persistence

## Использование

1. Откройте страницу с видео
2. Нажмите на иконку расширения в панели инструментов Firefox
3. Выберите скорость из пресетов или используйте кнопки +/- для точной настройки
4. Для редактирования пресета наведите на ячейку и нажмите иконку ✎
5. В модальном окне вы можете изменить скорость и назначить горячую клавишу

## Горячие клавиши

После назначения горячих клавиш в настройках пресетов, вы можете использовать их на любой странице с видео для быстрого переключения скорости.

## Области применения

- **Для этой вкладки** - скорость применяется только к текущей вкладке
- **Для этого домена** - скорость применяется ко всем вкладкам на текущем домене
- **Для всех вкладок** - скорость применяется ко всем вкладкам с видео

## Техническая информация

- **Версия манифеста**: 2 (Firefox)
- **Разрешения**: activeTab, storage, tabs, <all_urls>
- **Контентные скрипты**: Работает на всех страницах
- **Фоновый скрипт**: Обрабатывает горячие клавиши и настройки
- **API**: Использует `browser.*` API для совместимости с Firefox

## Примечания

- Работает с любым видео на любом сайте
- Поддерживает скорости от 25% до 1600%
- Настройки сохраняются автоматически
- Минимальное влияние на производительность
- Полная совместимость с Firefox (Manifest V2)
